# Merchant Feature

Reusable in-memory merchant data, smart search, state management, and cached brand icon rendering.

## Icon Attribution

Merchant brand icons are retrieved through the [Iconify API](https://iconify.design/docs/api/) from the [Simple Icons](https://simpleicons.org/) collection. Simple Icons is released under the [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/) public domain dedication.

Brand names and logos remain trademarks of their respective owners. Their inclusion does not imply endorsement, and consumers are responsible for following each brand owner's usage guidelines.
